# projeto-hugolopes-notebooklm-dio

Miniguia de Estudos com NotebookLM
Transformação Digital nas Empresas: SD-WAN, Cloud e Segurança Gerenciada

1. Contexto e Objetivos
Com o avanço da transformação digital, empresas de todos os portes buscam soluções capazes de melhorar conectividade, segurança e escalabilidade dos seus ambientes de TI.

Este estudo foi desenvolvido utilizando o NotebookLM como ferramenta de apoio à aprendizagem ativa, reunindo conteúdos sobre três pilares essenciais da infraestrutura moderna:

SD-WAN
Computação em Nuvem (Cloud)
Segurança Gerenciada
Objetivos

Ao final deste estudo pretendo:

Entender o funcionamento da tecnologia SD-WAN.
Conhecer os principais modelos de Cloud Computing.
Identificar os benefícios da Segurança Gerenciada.
Compreender como essas soluções podem apoiar empresas na transformação digital.
Criar um material de consulta rápida para futuras revisões.

3. Engenharia de Prompts
Prompt 1
Explique SD-WAN para um gestor que não possui conhecimento técnico.

Resultado
A IA respondeu utilizando linguagem simples, destacando redução de custos, gerenciamento centralizado e melhoria de desempenho.

Aprendizado
Prompts direcionados ao perfil do público geram respostas mais úteis.

Prompt 2
Compare MPLS e SD-WAN em uma tabela.

Resultado
Foi apresentada uma comparação clara entre custo, flexibilidade, segurança e escalabilidade.
Dificuldade Encontrada
Algumas respostas eram superficiais.
Ajuste Realizado
Compare MPLS e SD-WAN considerando custos, desempenho, segurança, escalabilidade e casos de uso empresariais.

Prompt 3
Quais riscos de segurança devem ser considerados em ambientes Cloud?

Resultado
Foram identificados:
Vazamento de dados
Configurações incorretas
Controle de acesso inadequado
Ataques cibernéticos

Prompt 4
Crie um plano de adoção de SD-WAN para uma empresa com 50 filiais.

Resultado
A IA gerou um roteiro estruturado com etapas de análise, implementação e monitoramento.
4. Cicatrizes e Troubleshooting

Problema 1
Respostas muito genéricas.

Solução
Adicionar contexto empresarial e objetivos específicos.

Problema 2
Comparações superficiais.

Solução
Solicitar critérios específicos para comparação.

Problema 3
Excesso de informações técnicas.

Solução
Pedir explicações voltadas para executivos ou tomadores de decisão.

5. Miniguia de Estudo
SD-WAN
O que é?

Tecnologia que utiliza software para gerenciar redes corporativas de forma inteligente.

Benefícios
Redução de custos
Maior disponibilidade
Melhor desempenho das aplicações
Gerenciamento centralizado
Desafios
Planejamento da migração
Integração com infraestrutura existente
Cloud Computing
O que é?

Disponibilização de recursos computacionais pela internet.

Modelos
IaaS

Infraestrutura como Serviço.

PaaS
Plataforma como Serviço.

SaaS
Software como Serviço.

Benefícios
Escalabilidade
Flexibilidade
Redução de investimentos em hardware
Segurança Gerenciada
O que é?

Serviços especializados para monitoramento e proteção de ambientes corporativos.

Benefícios
Monitoramento 24x7
Resposta rápida a incidentes
Conformidade regulatória
Redução de riscos
6. Glossário
SD-WAN

Rede definida por software para gerenciamento inteligente de conexões WAN.

MPLS
Tecnologia de rede privada amplamente utilizada por empresas.

Cloud Computing
Uso de recursos computacionais fornecidos pela internet.

Firewall
Sistema de proteção contra acessos não autorizados.

SOC
Security Operations Center.

Zero Trust
Modelo de segurança baseado em verificação contínua.

IaaS
Infrastructure as a Service.

PaaS
Platform as a Service.

SaaS
Software as a Service.
